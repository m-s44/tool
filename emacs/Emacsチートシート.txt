■Emacsチートシート

・修飾キー
Control => C-
Meta(windows:Alt Mac:option) => M-
Shift => S-
Super(MacのCommand) => s-

・特殊キー
SPC => Spaceキー

・共通
ターミナル(コマンドプロンプト)からの起動 => emacs -nw(emacs -no-window-system)

Emacs終了 => C-x C-c

前置引数
※C-u 8 AでAが8回入力される

操作のキャンセル => C-g
※予期せぬ動作をしたらとりあえず、C-g

システムコマンド => C-x

行番号表示 => M-x linum-mode

・ファイル移動系
上移動 => C-p
下移動 => C-n
右移動 => C-f
左移動 => C-b

1単語次に移動 => M-f
1単語前に移動 => M-b

行頭に移動 => C-a
行末に移動 => C-e

文頭に移動 => M-a
文末に移動 => M-e

1画面次に移動 => C-v
1画面前に移動 => M-v

ファイル先頭に移動 => M-<
ファイル末尾に移動 => M->

カーソル位置を起点に画面を書き直す => C-l
※1回目 => カーソル位置を中央に移動
※2回目 => カーソル位置を上端に移動
※3回目 => カーソル位置を下端に移動

行番号移動 => M-g M-g(ミニバッファに移動、指定の行番号を入力、Enter)

検索 => C-s(ミニバッファに移動、検索したい文字列を入力、Enter)
    　  C-r(逆方向に検索)
       検索後、C-sで次の一致場所に、C-rで前の一致場所に移動




・ファイル編集系
編集の取り消し(アンドゥ) => C-x u、C-/、C-_

カーソル位置から行末まで消去 => C-k
直前の消去を取り消し => C-y

マークとリージョン => C-SPC

コピー => M-w

カット => C-w

ペースト => C-y

対話置換 => M-%(ミニバッファに移動、検索する文字列入力、置換する文字列入力、一致すればyで置換、?でヘルプを表示、nでスキップ)

一括置換 => M-x replace-string(ミニバッファに移動、検索する文字列を入力、置換する文字列を入力、Enter)

・ファイル関係
ファイル開く => C-x C-f

ファイルの保存 => C-x C-s

別名でファイルを保存する => C-x C-w

ファイルに別ファイルを挿入する => C-x i

ファイル(バッファ)を切り替える => C-x b

ファイル(バッファ)を消去する => C-x k

・ウィンドウ系
ウィンドウを横分割する =>C-x 2

ウィンドウを縦分割する => C-x 3

ウィンドウを移動する => C-x o

ウィンドウを１つ増やす => C-h k C-f

ウィンドウを閉じる => C-x 1、C-x 0

・ディレクトリ操作
ディレクトリを開く => C-d


・キーボードマクロ
C-x ( => マクロスタート
繰り返したい操作
C-x ) => マクロを記録

C-x e => マクロを実行

M-x name-last-kbb-macro マクロ名 => 直近のマクロに名前をつける
M-x マクロ名

・表示の変更
文字の表示サイズを大きくする => C-x C-+

文字の表示サイズを小さくする => C-x C--

文字の表示サイズを元に戻す => C-x C-0

・ヘルプ
C-h 引数 => 引数に応じた、ヘルプを表示する
