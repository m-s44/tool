■Emacsチートシート

・修飾キー
Control => C-
Meta(windows:Alt Mac:option) => M-
Shift => S-
Super(MacのCommand) => s-


・共通
ターミナル(コマンドプロンプト)からの起動 => emacs -nw(emacs -no-window-system)

Emacs終了 => C-x C-c

前置引数 => C-u
※C-u 8 AでAが8回入力される

操作のキャンセル => C-g
※予期せぬ動作をしたらとりあえず、C-g

システムコマンド => C-x



・ファイル移動系
上移動 => C-p
下移動 => C-n
右移動 => C-f
左移動 => C-b

1単語次に移動 => M-f
1単語前に移動 => M-b

行頭に移動 => C-a
行末に移動 => C-e

文頭に移動 => M-a
文末に移動 => M-e

1画面次に移動 => C-v
1画面前に移動 => M-v

ファイル先頭に移動 => M-<
ファイル末尾に移動 => M->

画面を書き直す => C-l
※1回目 => カーソル位置を中央に移動
※2回目 => カーソル位置を上端に移動
※3回目 => カーソル位置を下端に移動



・ファイル編集系
編集の取り消し => C-x u

カーソル位置から行末まで消去 => C-k
直前の消去を取り消し => C-y


・ファイル関係

ファイル開く => C-x C-f

ファイルの保存 => C-x C-s
