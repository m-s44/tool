■ノーマルモード
・カーソル移動

k => 上に移動

j => 下に移動

h => 左に移動

l => 右に移動



・行検索
f{文字} => 指定した{文字}を行末尾方向に検索して、
           マッチしたら、カーソル位置をマッチした位置まで移動する(行単位)

F{文字} => 指定した{文字}を行先頭方向に検索して、
           マッチしたら、カーソル位置をマッチした位置まで移動する(行単位)

t{文字} => 指定した{文字}を行末尾方向に検索して、
           マッチしたら、カーソル位置をマッチした{文字}の一文字前に移動する(行単位)

T{文字} => 指定した{文字}を行先頭方向に検索して、
           マッチしたら、カーソル位置をマッチした{文字}の一文字前に移動する(行単位)

; => 実行した直前の行検索を繰り返す(行単位)

, => 実行した直前の行検索を戻す(行単位)



・ファイル内検索
/文字列 => 入力した文字列をファイル内検索し、マッチした位置までカーソルを移動する

* => カーソル位置にある単語をファイル内検索する、マッチした位置までカーソルを移動する

n => 「/文字列」と「*」で検索したことを次にマッチする位置までカーソルを移動する。

N => 「/文字列」と「*」で検索したことを前にマッチした位置までカーソルを移動する。



・ファイル編集系
. => 直前に行った編集を繰り返す

u => 直前の編集を元に戻す

■挿入(インサート)モード
s => カーソル位置の文字を削除して、挿入モードに切り替える
