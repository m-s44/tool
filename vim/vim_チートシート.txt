■Vimの基本コマンド系
:Tutorial => vimのチュートリアルを開く

:help 設定またはコマンド => 設定のヘルプを開く

:set 設定? => 設定の現在の状態を調べる

:set コマンド=○○、:set 設定 => 設定のパラメータを設定する


■ノーマルモード
・カーソル移動

k => 上に移動

j => 下に移動

h => 左に移動

l => 右に移動

w(W) => 前方(右方向)に移動

e(E) => 前方(右方向)に移動(単語終わりまで)

b(B) => 後方(左方向)に移動

gg => ファイル先頭に移動

G => ファイル末尾に移動

^ => 行頭(最初の文字まで)移動

0 => 行頭に移動

$ => 行末に移動

zz => 画面を中央に再描画する(カーソルが中央に移動)

・行検索
f{文字} => 指定した{文字}を行末尾方向に検索して、
           マッチしたら、カーソル位置をマッチした位置まで移動する(行単位)

F{文字} => 指定した{文字}を行先頭方向に検索して、
           マッチしたら、カーソル位置をマッチした位置まで移動する(行単位)

t{文字} => 指定した{文字}を行末尾方向に検索して、
           マッチしたら、カーソル位置をマッチした{文字}の一文字前に移動する(行単位)

T{文字} => 指定した{文字}を行先頭方向に検索して、
           マッチしたら、カーソル位置をマッチした{文字}の一文字前に移動する(行単位)

; => 実行した直前の行検索を繰り返す(行単位)

, => 実行した直前の行検索を戻す(行単位)

ga => カーソル文字の数値コードを得る

r => 置換モードに切り替える(カーソルの1文字だけ)

R => 置換モードに切り替える(入力しただけ置換する)
※入力中にBackspaceをタイプすれば、置換前の文字に戻る

gR => 仮想置換モードに切り替える

Ctr+g => 開いているファイルパスと現在行を表示する

・ファイル内検索
/文字列 => 入力した文字列をファイル内検索し、マッチした位置までカーソルを移動する

* => カーソル位置にある単語をファイル内検索する、マッチした位置までカーソルを移動する

n => 「/文字列」と「*」で検索したことを次にマッチする位置までカーソルを移動する

N => 「/文字列」と「*」で検索したことを前にマッチした位置までカーソルを移動する

% => 対応する括弧の検索

:s/検索文字/置換文字 => 現在行の最初にヒットする検索文字を置換文字に置換する

:s/検索文字/置換文字/g => 現在行の検索文字を置換文字に全て置換文字に置換する

:%s/検索文字/置換文字/g => ファイル内の検索文字を全て置換文字に置換する(確認なし)

:%s/検索文字/置換文字/gc => ファイル内の検索文字を置換文字に置換する(1文字ずつ確認する)

・ファイル編集系
. => 直前に行った編集を繰り返す

u => 取り消し

Ctr+r => 取り消しの取り消し

d => 行を削除
dw => 前方単語削除

Ctr+a => 数値を加算する(数値を入力後に実行した場合はその数だけ加算される)

Ctr+x => 数値を減算する(数値を入力後に実行した場合はその数だけ減算される)

yy => 行をヤンク(コピー)する

yw => １単語をヤンク(コピー)する

p => ヤンク(コピー)した文字を貼り付ける

s => カーソル位置の文字を削除して、挿入モードに切り替える

i => カーソル位置の1文字前に、挿入モードに切り替える

I => 行頭に挿入モードに、切り替える

a => カーソル位置の1文字後に、挿入モードに切り替える

A => 行末に挿入モードに切り替える

o => 1行をあけて、挿入モードに切り替える

O => 1行前をあけて、挿入モードに切り替える

r => カーソル位置の文字を置換する

R => カーソル位置の文字を複数文字置換する

v => ビジュアルモードに切り替える

V => 行のビジュアルモードに切り替える

Ctr+v => ブロック単位(矩形選択)のビジュアルモードに切り替える

gv => 直前のビジュアルモードに切り替える

:e ファイル名 => 新規ファイルを開く

:e . => カレントディレクトリを開く


・タブ系
:tabnew => 新しいタブを開く

gt => 次のタブに移動する

gT => 前のタブに移動する

・外部コマンド系
:!コマンド => 外部コマンド(シェルコマンド)を実行する




■挿入(インサート)モード
Ctr+h => 直前の1文字を削除(BackSpace)

Ctr+w => 直前の１単語を削除する

Ctr+u => 行頭まで削除する

Esc、Ctr+[ => ノーマルモードに切り替え

Ctr+o => 挿入ノーマルモードに切り替える(挿入モードの状態で一度だけノーマルモードに切り替えられる)

Ctr+r= => Expressionレジスタ

Ctr+n、Ctr+p => キーワード補完

■ビジュアルモード
Esc、Ctr+[ => ノーマルモードに切り替える

v => 文字指向のビジュアルモードに切り替える

V => 行指向のビジュアルモードに切り替える

Ctr+v => ブロック指向のビジュアルモードに切り替える(ブロック指向のビジュアルモードの場合はノーマルモードに切り替える)

U => ビジュアルモード中だと小文字を大文字に切り替える

u => ビジュアルモード中だと大文字を小文字に切り替える

o => ビジュアルモードで選択中のカーソルの始点と終点を切り替える

※※※※※※※※※※※※※※※※※※※
下記ノーマルモードのキーも使えます
j、k、l、h
f{char}、,、;
